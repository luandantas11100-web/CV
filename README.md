📄 Currículo Online - Luan Dantas Melo
Este é um projeto de Currículo Online (Single Page) desenvolvido em HTML5 puro. O objetivo é apresentar de forma clara e estruturada as informações profissionais, acadêmicas e habilidades técnicas de um estudante de Ciência da Computação.

🚀 Tecnologias Utilizadas
O projeto foca na utilização de HTML Semântico para garantir melhor acessibilidade e SEO (Otimização para motores de busca).

HTML5: Estruturação semântica (<main>, <section>, <address>).

Meta Tags: Configuração de viewport para dispositivos móveis e descrições para buscadores.

Acessibilidade: Uso correto de tags de cabeçalho e links de contato.

🛠️ Habilidades Técnicas
O currículo destaca competências tanto em Frontend quanto em Backend e Linguagens Funcionais:

Frontend: HTML, CSS, Javascript, Responsive Web Design.

Backend: Java, Spring Boot.

Linguagens Acadêmicas/Funcionais: Haskell.

📂 Estrutura do Projeto
index.html: Arquivo principal contendo toda a estrutura do currículo.

imagens/: Pasta destinada ao armazenamento de ativos visuais (como o favicon).

📝 Observações sobre o Código
Durante o desenvolvimento, foram aplicados conceitos importantes discutidos anteriormente:

Tag <address>: Utilizada corretamente na seção "Across the internet" para agrupar os links de contato profissional.

Meta Tags de SEO: Inclusão de description para ajudar na indexação por motores de busca.

Favicon: Configuração de ícone personalizado na aba do navegador.
